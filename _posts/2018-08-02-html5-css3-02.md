# CSS로 문서 작성하기

CSS 는 HTML 문서에 **장식** 도는 **레이아웃**을 설정하기 위한 언어입니다


## HTML 언어에 CSS 적용방법

1. **inline** : h1 style="속성:값;"
2. **head**의 **style** 에서 **내부참조**
3. **외부참조** : 1) \<link href = ".css"\>  2) \<style\> @import url(외부 css경로);
