# Django Setting.py

**Html5의 기초 -> Css 스타일 기초 -> JavaScript 기본기능 -> Vue 패키지 보완** 순서대로 client 에서 조작 가능한 부분들을 다뤄보았다.

**필요로 하는 부분을 어떻게 구현할 것인가를** 직관적 빠르게 파악하는 능력을 갖추기 위해선 **전체적인 Process**를 이해하고 이에 필요한 내용들을 잘 정리하는게 중요하다.

Django Setting.py 에서는 2가지 중요한 과정이 있는데 하나는 **개발단계**, 나머지는 **배포단계**이다. 이번장에서는 개발단계에서 필수적 추가할 내용을 정리 및 Link 로써 정확하고 빠른 작업에 도움이 되기위한 목적으로 작성을 하였다.
