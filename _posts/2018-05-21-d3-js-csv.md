# D3.js

## CSV 외부 데이터 불러오기

`d3.csv()` 를 사용해서 자료 불러오기 

```javascript
d3.csv("mydata.csv", 
    function(error, data){
       var dataSet = [ ];
       for(var i=0; i<data.length; i++){
          dataSet.push(data[i].item1);}
  
    d3.select("#myGraph") 
      .selectAll("rect")  // rect 요소 지정
      .data(dataSet)  // 데이터를 요소에 연결
      .enter()  // 데이터 개수만큼 반복
    });
```

**data.length :** 데이터 줄 수(index)만큼 반복
