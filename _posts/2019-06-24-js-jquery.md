# Jquery 시작하기

## selector

웹페이지 **DOM** 중 Jquery 함수를 적용할 대상을 특정하는 방법 입니다. 아래의 내용처럼 대상을 `$(document).ready(function(){ })` 에서 시작하여 특정할 수 있지만, 번거로운 부분이 많아서 `( $(function(){ }))` 와 같은 축약형을 주로 사용 합니다.

```javascript
$('#선택자.선택자').css('background','pink');

$(function() {
  $('태그').fadeOut(3000);
});

$(document).ready(function(){
  $('#sample1').fadeOut(3000);
});
```

이때 선택자 및 속성값을 **String** 으로 사용자가 직접 입력하는 만큼 오탈자로 인한 오류에 주의해서 작업을 진행 합니다.

그리고 앞에서 **CSS** 선택자 문법의 **#, ., >, ~, nth-child(n)** 등 다양한 선택자를 활용하면 보다 손쉽게 대상을 특정할 수 있습니다
