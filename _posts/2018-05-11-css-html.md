# CSS Style

## CSS 기본문법 

1. **head 태그** 속에 **Style 태그**로 CSS 문법들을 기록한다
2. **html 태그** 속에 **Style 속성값**으로 CSS 문법을 기록한다

```html
<head>
    <style type="text/css">
    a {
        color:black;
    }
    </style>
</head>
<body>
    <a href="#"  style="color:red">Css</a>
</body> 
</html>
```

**style 태그**는 대상을 지정하고 **{} 중괄호**로 내용을 기록한다, **style 속성**은 **html 문법형식으로 기록**하는 차이를 갖는다
