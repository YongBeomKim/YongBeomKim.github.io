# 웹 표준 디자인 강좌

## 레이아웃의 종류 

1. **Fixed Layouts** : 고정값으로 너비를 지정
2. **Liquid Layouts** : 퍼센트 비율로 너비를 지정
3. **Flexible Layouts** : 1, 2 번의 장점을 활용하여 레이아웃을 구성 
4. **Responsive Layouts** : 반응형 레이아웃 

**Liqiuid Layout :** 크기가 극단적으로 크거나 좁은 경우에는 가독성이 떨어지는 문제가 있다
